## Skills

The system-reminder contains the full list of available skills and guidance on when to use them. Read the skill list at the start of every session and invoke matching skills as appropriate.

> **EMERGENCY RULE (MUST follow):** If the system-reminder does NOT contain the skill list, something is internally broken. Do NOT proceed with any work. Immediately terminate the conversation and notify the user that an emergency has occurred (skills failed to load). Proceeding without skills loaded would cause catastrophic failures.
